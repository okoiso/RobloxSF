# Roblox Server Finder

A desktop application built in Python to find Roblox servers with lower latency using real-time API data.

# Description

Roblox Server Finder allows users to search public servers using a Place ID and rank them based on performance (ping and FPS). The goal is to help users join more stable and less laggy servers.

# Features
Search servers by Place ID
Fetch data from Roblox API
Rank servers using a performance score
Display ping, FPS, and player count
Join servers directly through the Roblox client
GUI with status updates and progress feedback
Background execution to prevent UI freezing
How to Use
Open the application
Enter a valid Place ID
Click "Search"
Wait for the servers to load
Select a server and click "Join"
Executable

You can use the application without installing dependencies by downloading the .exe file available in the releases section.
